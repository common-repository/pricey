誰でも簡単に投稿本文でAmazon, 楽天, Yahoo!のアフィリエイトリンクを挿入できます。デザイン不要で、かわいいコンポーネントを自動生成。アフィリエイトリンクの管理もこれ一つで完結。

== Description ==
プライシーは、WordPress用の革新的なアフィリエイトリンク挿入プラグインで、Amazon（アマゾン）、楽天、Yahoo!ショッピングのアフィリエイトリンクを投稿本文に簡単に挿入できるだけでなく、デザインの知識がないユーザーでも魅力的なコンポーネントを自動生成し、サイトの見た目を向上させることができます。

プライシーの特徴は、アフィリエイトリンクの効率的な管理と自動価格比較機能に加え、新たに商品の価格推移を自動表示する機能が追加されました。
この機能により、コンポーネント作成時に、自動的にその商品の価格変動の履歴を表示し、サイト訪問者に価格の動きを一目で確認できるようになります。これは、購入を検討しているユーザーにとって非常に貴重な情報を提供し、購入決定を助けます。

このプラグインは、特にアフィリエイトマーケティングを行っているブロガーやオンラインショップのオーナーにとって、サイト訪問者に最適な価格オプションを提供し、アフィリエイト収益を最大化することが可能になります。

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “My Custom Style Css Manager”.
3. Select the “My Custom Style Css Manager” plugin and click “Install”.
4. Activate the plugin.

OR…

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
